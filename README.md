#  Quản Lý Kho Vật Tư

Ứng dụng Android hỗ trợ **quản lý kho vật tư** với các chức năng như:
- Quản lý sản phẩm
- Quản lý khách hàng
- Quản lý nhà cung cấp
- Quản lý nhân viên
- Theo dõi lịch sử nhập / xuất kho
- Báo cáo tài chính
- Báo cáo tồn kho

---

##  Công nghệ sử dụng

###  Android
- Java
- Android Studio
- RecyclerView
- BottomSheetDialog
- MPAndroidChart (Biểu đồ thống kê)

###  Backend
- Node.js
- Express.js
- REST API

###  Database
- MySQL
